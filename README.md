# 🎥 CINEMATIC TICKETING SYSTEM 

## 📌 Title
Movie Ticket Booking System

## 📚 Overview
 The Movie Ticket Booking System is a console-based application that allows users to browse movies, select showtimes, choose seats, and book tickets efficiently.
 It helps reduce long queues by providing a fast and organized booking process while ensuring that seats are not double-booked.

## 🤖 Project Structure + How to Run the Program
### 📝 Requirements
- C++ compiler (e.g., g++)
   
### 🏃‍♂️ How to Run
- main.cpp



## ✨ Features
- 🎥 Movie Listing
- ⏰ Showtime Selection
- 💺 Seat Availability Checking
- 🎟️ Ticket Booking
- ⏳ Add User To Queue
- 🚀 Serve Next User
- 🔄 Undo Last Booking
- 🗓️ Sorted Booking Seats
- 🛂 Admin Dashboard
